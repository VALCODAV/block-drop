About Block-Drop

Block-Drop is a multiplayer tetris-like game. Watch an opponent play by selecting them from the player list. Attack an opponent by selecting them from player list and then clearing a row. Fight for the highest score by clearing rows and sabotaging your opponents.

Keyboard Controls

Move horizontally
Rotate
Increase block speed
Drop block
Pause the game
Move forwards and backwards through the list of players

Mouse/Touch Controls

Drag blocks horizontally or downwards
Two-finger tap tap to rotate blocks
Double-tap to drop blocks
Tap opponent's name to select as target
Swipe upwards to pause the game
